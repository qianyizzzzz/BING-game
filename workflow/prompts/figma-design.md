```text
请根据以下资料设计这个游戏界面：

- workflow/docs/00-game-pillars.md
- workflow/docs/01-visual-bible.md
- workflow/tasks/TASK-XXX.md

目标界面：
[填写界面名称]

请在 Figma 中提供：
1. 默认状态
2. hover 状态
3. pressed 状态
4. selected / controller focus 状态
5. disabled 状态
6. error 或 empty 状态，如果适用
7. 小屏幕布局
8. 颜色、字体、间距标注
9. 组件命名

设计要求：
- 视觉必须支持 Game Pillars。
- UI 不要像普通网页后台。
- 不要只做一张静态漂亮图，必须包含状态。
- 重要操作必须有清晰反馈。
- 文本必须在小屏幕下可读。

完成后请给出 frame 链接，并说明每个 frame 对应的状态。
```
