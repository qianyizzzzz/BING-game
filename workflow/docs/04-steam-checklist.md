# Steam Checklist

这份清单用于准备未来 Steam 上架。它不是一次性完成的，建议从垂直切片阶段就开始维护。

## Store Basics

- [ ] Game title
- [ ] Short description
- [ ] Long description
- [ ] Genre and tags
- [ ] Developer name
- [ ] Publisher name if different
- [ ] Supported languages
- [ ] Planned release window
- [ ] Pricing direction

## Store Assets

- [ ] Header capsule
- [ ] Small capsule
- [ ] Main capsule
- [ ] Vertical capsule
- [ ] Library capsule
- [ ] Library hero
- [ ] Logo asset
- [ ] At least 5 screenshots
- [ ] Trailer
- [ ] Animated GIFs or short clips for social posts

## Build Readiness

- [ ] Windows build runs from a clean install.
- [ ] No editor-only paths remain.
- [ ] Save location is correct.
- [ ] Settings persist.
- [ ] Fullscreen, windowed, and borderless behavior is acceptable.
- [ ] Audio volume sliders work.
- [ ] Crash logs or bug reporting path exists.

## Input

- [ ] Keyboard and mouse supported.
- [ ] Rebinding plan decided.
- [ ] Controller support evaluated.
- [ ] Steam Deck readability checked.
- [ ] UI focus states are visible.

## Demo

- [ ] Demo has a clear start.
- [ ] Demo has a clear ending.
- [ ] Demo does not expose unfinished production menus.
- [ ] Demo can collect feedback.
- [ ] Demo has a known build number.

## Review and Release

- [ ] Steamworks account and app created.
- [ ] Steam Direct fee handled.
- [ ] Store page prepared.
- [ ] Store page submitted for review.
- [ ] Build uploaded.
- [ ] Build submitted for review.
- [ ] Coming Soon page timing planned.
- [ ] Launch checklist created.

## Marketing

- [ ] One-sentence pitch
- [ ] 3-5 key selling points
- [ ] Press kit folder
- [ ] Social post screenshots
- [ ] Trailer script
- [ ] Demo announcement plan
- [ ] Wishlist CTA copy
